Feature Guides

Learn about the features of DataHub.

🗃️ Ask DataHub

1 items

🗃️ Agent Context Kit

8 items

📄️ Agents

Starting in DataHub Cloud v1.0.1, Agents is in Private Beta and available only on DataHub Cloud. To enable this feature for your organization, contact the DataHub team.

📄️ Analytics Agent

An open-source agent that lets you ask data questions in plain English and get SQL, results, and charts back — grounded in your DataHub catalog. Apache 2.0, bring your own LLM. Read the announcement blog post.

🗃️ Assertions (Data Quality)

11 items

📄️ Data Access Workflows

Use DataHub Cloud Data Access Workflows to centralize approval processes for data access requests with reviewer routing.

📄️ Data Access Roles

This feature is currently in Private Beta.

📄️ Applications

DataHub Cloud Applications group related assets by business purpose, providing an experimental way to model software systems and services.

🗃️ Automations

7 items

📄️ Business Attributes

Business Attributes in DataHub bundle business names, descriptions, tags, glossary terms, and data quality rules into reusable units.

📄️ Business Glossary Guide

Build a Business Glossary in DataHub to define a shared vocabulary of business terms and link them to physical data assets.

🗃️ Compliance Forms

4 items

📄️ Context Documents

Capture organizational tribal knowledge in DataHub with Context Documents — runbooks, FAQs, decision logs, and process guides.

📄️ Asset Summaries

Customize the Summary tab for Domains, Data Products, and Glossary Terms in DataHub to create curated discovery experiences.

📄️ Home Page

Build personalized DataHub home pages with modular widgets to surface the most relevant data assets and information for each user.

📄️ Data Contracts Monitoring

Define Data Contracts in DataHub Cloud Observe as agreements between producers and consumers covering schema, freshness, and quality.

📄️ Data Health Dashboard

Use the Data Health Dashboard in DataHub Cloud Observe to monitor assertion status, freshness, and data quality across assets.

📄️ Data Products

Group related Tables, Dashboards, Charts, and Pipelines into Data Products in DataHub to support data mesh ownership and discovery.

📄️ Dataset Usage and Query History

Dataset Usage & Query History can give dataset-level information about the top queries which referenced a dataset.

📄️ Domains Overview

Use DataHub Domains to group related data assets into curated, top-level collections owned by business units or teams.

📄️ File Upload and Download in Documentation

Attach images, diagrams, and supporting files to asset and column documentation in DataHub to enrich your data catalog.

📄️ Incidents Overview

This page provides an overview of working with the DataHub Incidents API.

📄️ Ingestion

Set up no-code metadata ingestion in DataHub through the UI to automatically pull metadata from your data sources on a schedule.

📄️ Ingestion Security

DataHub supports three ways to ingest metadata. They differ primarily in where credentials are stored and what network access is required.

🗃️ Lineage

4 items

🗃️ Logical Models

2 items

📄️ Metadata Tests

DataHub includes a highly configurable, no-code framework that allows you to configure broad-spanning monitors & continuous actions

📄️ MCP Server

The DataHub MCP Server lets AI agents query DataHub metadata via the Model Context Protocol to find assets and traverse lineage.

📄️ Ownership

Define custom Ownership Types in DataHub beyond the built-in Technical Owner, Business Owner, and Data Steward roles.

📄️ Policies

Access Policies define who can do what to which resources. In conjunction with Roles, Access Policies determine what users are allowed to do on DataHub.

📄️ Posts

Create Posts in DataHub to share announcements, links, and updates on the home page and across surfaces of the application.

2 items

📄️ Search Overview

Use the DataHub search bar to find datasets, columns, dashboards, charts, and pipelines across your data ecosystem.

📄️ Search Access Controls

Restrict which entities users can discover in DataHub search results using Search Access Controls and View Entity permissions.

📄️ Service Accounts

Available starting in DataHub Cloud v0.3.17, DataHub Core v1.4.0.

📄️ Subscriptions & Notifications

DataHub's Subscriptions and Notifications feature gives you real-time change alerts on data assets of your choice.

📄️ Sync Status

When looking at metadata in DataHub, it's useful to know if the information you're looking at is relevant.

📄️ Tags Overview

Use Tags in DataHub as informal, loosely-controlled labels to classify and search for datasets, schemas, and containers.

📄️ Views

Views let you save a set of filters and reuse them across DataHub. When you activate a View, search results, browse pages, recommendations, and other asset lists are automatically narrowed to only the matching assets.
