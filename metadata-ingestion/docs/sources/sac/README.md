## Overview

SAP Analytics Cloud is a business intelligence and analytics platform. Learn more in the [official SAP Analytics Cloud documentation](https://www.sap.com/products/technology-platform/analytics-cloud.html).

The DataHub integration for SAP Analytics Cloud covers BI entities such as dashboards, charts, datasets, and related ownership context. Depending on module capabilities, it can also capture features such as lineage, usage, profiling, ownership, tags, and stateful deletion detection.
