Lineage Client
==============

The DataHub Lineage Client provides a client for adding and retrieving lineage information from DataHub.

If you're looking for higher-level introduction to adding and getting lineage using the SDK, see the [lineage guide](../../docs/api/tutorials/lineage.md).
